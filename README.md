# 雅思词汇闪卡 (IELTS Vocabulary Flashcards) 🚀

一个简洁、优雅且高效的雅思词汇背诵工具。专为备考雅思的同学设计，基于艾宾浩斯遗忘曲线原理，通过闪卡（Flashcards）形式帮助你快速斩获核心词汇。

本项目是 **Vibe Coding** 的典型实践：由 AI 辅助进行快速迭代开发。它打破了传统背词软件的臃肿，回归最纯粹的闪卡记忆逻辑。

**👉 [在线使用](https://kinsung97.github.io/IELTS-Vocab/)** | **无需安装，打开即用！**

## ✨ 核心功能

### 🎯 智能背诵
| 功能 | 说明 |
|------|------|
| 🎴 **闪卡模式** | 隐藏释义，强制回忆，点击翻转查看 |
| 🧠 **艾宾浩斯遗忘曲线** | 根据熟练度自动安排复习：生疏(1天)、模糊(2天)、熟悉(4天)、简单(7天) |
| ⭐ **生疏词优先** | 标记为"生疏"的单词会在第二天优先出现，确保薄弱点及时巩固 |
| 🔊 **智能发音** | 集成 TTS 技术，支持自动/手动播放单词发音，可在设置中开关 |

### 📊 进度管理
| 功能 | 说明 |
|------|------|
| 📈 **可视化进度** | 总进度 + 各分类进度条，直观掌握学习情况 |
| ✅ **已背/未背单词列表** | 查看所有单词，支持按时间、熟练度筛选 |
| 📝 **修改熟练度** | 在已背单词列表中点击熟练度标签，随时更正记忆程度 |
| 📅 **每日重置** | 跨天自动重置"今日已背"计数，但保留所有学习记录 |

### 🎨 个性化设置
| 功能 | 说明 |
|------|------|
| 🌈 **主题色切换** | 支持紫罗兰、天空蓝、薄荷绿、活力橙四种主题 |
| 🌙 **夜间模式** | 深度适配暗色主题，深夜刷词不刺眼 |
| 📚 **分类词库** | 支持全库混背或按分类精准突破 |

### 💾 数据安全
| 功能 | 说明 |
|------|------|
| 🔒 **本地存储** | 所有数据保存在浏览器本地，**不上传任何数据到服务器**，保护隐私 |
| 💾 **进度备份** | 支持 JSON 格式的进度导出/导入，换设备无缝衔接 |
| 📊 **Excel 导出** | 已背单词可一键导出为 CSV 格式，方便复盘分析 |

## 📖 使用指南

### 快速开始
1. 访问 **https://kinsung97.github.io/IELTS-Vocab/**
2. 点击"开始今天的挑战"进入背诵
3. 看到单词 → 尝试回忆 → **点击卡片**查看释义
4. 根据记忆程度选择：**生疏(1d)** / **模糊(2d)** / **熟悉(4d)** / **简单(7d)**

### 进阶功能
- **切换主题色**：侧边栏 → 🎨 主题色
- **开关自动发音**：侧边栏 → 🔊 自动发音
- **查看进度**：侧边栏 → 📈 学习进度
- **修改熟练度**：侧边栏 → ✅ 已背诵单词 → 点击熟练度标签
- **备份数据**：侧边栏 → 📥 备份进度
